# Building and Running

## Build the program

```
make tp1
```

## Build the documentation

```
make doc
```

```
make doc-extensive
```

## Build the program and the documentation

```
make
```

## Download the media files

```
make media
```

## Run the program

```
./tp1
```

```
make run
```

## Open the documentation

```
chromium-browser doc/html/index.html
```

## Clean build files

```
make clean
```

## Create release archive

```
make release
```
